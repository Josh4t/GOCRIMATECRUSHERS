What is Html

Html stands for YHYperextMarkupLanguage. Its a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

Html consists of elements that describe different types of content: paragraphs, links, headings, images, videos, etc. web browsers understands Html will render Html codes as websites.

In Html, each elements is made up of 3 parts:

  1. The Opening tag
  2. The Closing Tag
  3. The Actual Element
